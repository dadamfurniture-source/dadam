# 다담 캐비넷 설계 가이드

## 개요
다담 캐비넷 시스템은 세 가지 주요 가구 타입을 지원합니다:
- 싱크대 (Sink Cabinet)
- 붙박이장 (Wardrobe)
- 냉장고장 (Fridge Cabinet)

## 싱크대 (Sink Cabinet)

### 구조
```
┌─────────────────────────────────┐
│           상몰딩                 │
├─────────────────────────────────┤
│           상부장                 │  ← 후드, 수납장
├─────────────────────────────────┤
│           상판 영역              │
├─────────────────────────────────┤
│           하부장                 │  ← 개수대, 쿡탑, 수납장
├─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┤
│   다리발 + 걸레받이              │
└─────────────────────────────────┘
```

### 도어 규칙
- 도어 개수: `Math.max(1, Math.round(모듈너비 / 450))`
- 도어 간격: 3mm
- 도어 버튼 클릭시 아이콘 숨김 (키큰장 제외)

### 걸레받이
- 높이: 다리발 높이 - 5mm
- 너비: 하부장 너비 합계 (최대 2400mm)
- 도어 표시시 자동 생성

## 붙박이장 (Wardrobe)

### 모듈 타입
| 타입 | 설명 | 분할 |
|------|------|------|
| short | 짧은옷(2단) | O |
| long | 긴옷(1단) | X |
| shelf | 선반형 | O |

### 자동배치 규칙
- 2통: 짧은옷(2단), 긴옷(1단)
- 3통: 짧은옷(2단), 짧은옷(2단), 긴옷(1단)
- 4통: 짧은옷(2단), 짧은옷(2단), 긴옷(1단), 선반형

### 손잡이 타입
- 일반 (bar): 기존 분배규칙
- 스마트바 (smartbar): 도어 균등분배 (목표 480mm)

## 냉장고장 (Fridge Cabinet)

### 구조
```
┌─────────────────────────────────┐
│           상몰딩                 │
├─────────────────────────────────┤
│           상부장                 │  ← 도어 1개/모듈
├─────────────────────────────────┤
│                                 │
│   냉장고 공간 / 키큰장 / 홈카페장  │
│                                 │
├─────────────────────────────────┤
│           좌대                   │
└─────────────────────────────────┘
```

### 도어 규칙
| 영역 | 도어 개수 |
|------|----------|
| 상부장 | 1개 고정 |
| 키큰장 (중간장/하부장) | 너비 기준 분배 (450mm) |
| 홈카페장 (하부장) | 너비 기준 분배 (450mm) |

### 도어 색상
- `getDoorColor()` 함수 사용
- 색상: 화이트, 그레이, 베이지, 네이비, 블랙
- 도어 간격: 3mm

## 공통 규칙

### 마감재
| 타입 | 기본 너비 |
|------|----------|
| 휠라 (Filler) | 60mm |
| 몰딩 (Molding) | 60mm |
| EP | 20mm |
| 없음 (None) | 0mm |

### 치수 계산
- 유효공간 = 전체너비 - 좌측마감 - 우측마감
- 모듈 높이 = 전체높이 - 상몰딩 - 좌대
